## 技术栈：

> react16
> react-router4
> react-redux
> babel7
> webpack4
> express

## 开发环境：

```js

npm install OR yarn install 
npm run dll ------- 分离依赖库
npm run build ------- 打包文件
node index ------- 开起服务

```
